# L’Arbre des Décisions Existentielles de Maelys

Maelys doit construire un BST représentant des choix de vie codés par des chiffres :

- 50 : “Faire ses devoirs”
- 30 : “Regarder une série”
- 20 : “Manger”
- 40 : “Dormir”
- 70 : “Coder un projet perso”
- 60 : “Lire un livre”
- 80 : “Commander un café”

Elle doit :

- construire le BST
- afficher les choix dans l’ordre croissant
- vérifier si “Dormir” (40) existe réellement dans ses options (spoiler : rarement)
